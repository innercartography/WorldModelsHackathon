01 — The Concept

Not a visualization.
A navigation system.

Most knowledge graphs show you information. This one takes you there.

The graph floats in space around you. Each node is a place, a story, a moment in time — anchored to real community knowledge, real historical assets, real coordinates on earth.

You reach out. You pinch a node. The graph dissolves. The world opens.

What was data becomes environment. The knowledge graph is the map. The world model is the territory.

02 — Three Layers of Reality
THE GRAPH
THE WORLD
THE MEMORY
LAYER 01
THE GRAPH
A force-directed 3D knowledge graph already live in the browser. Nodes are concepts, places, people, moments. Edges are relationships. Three layout modes — centralized, decentralized, distributed. This is your compass inside the experience.
LAYER 02
THE WORLD
World Labs Marble API turns community assets — old photographs, spatial scans, oral histories — into navigable 3D environments. Each node in the graph has a world behind it. The world model is the territory the graph describes.
LAYER 03
THE MEMORY
Community knowledge as infrastructure. DVV — Dunsmuir Visionary Village — is the first shard. A bioregional community's history, land, and future as navigable space. Real places. Real stories. Real people who lived them.
03 — The Stack
Live now
Three.js Graph
DVV spatial knowledge graph running on Vercel. Multiple layout modes, full node interaction, world.json data structure. This is the foundation — already working.
Building
WebXR Layer
WebXR session entry on top of existing Three.js. Hand tracking via WebXR Hand Input API. Pinch gesture for node selection. Browser-native — opens on any XR device via URL, no install.
Integrating
World Labs Marble
Node selection triggers Marble API call. Historical assets feed in. Navigable world generates around the user. The hackathon provides API access. This is the world model layer.
Stretch goal
Societies of Thought
Multiple AI perspectives inhabiting the graph as persistent lenses — each node weighted by consensus and conflict between viewpoints. The graph topology reflects the epistemic state of the community.
